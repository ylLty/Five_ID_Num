## 原理说明

```纯文本
针对已在[神人列表]中的id:

	直接给102%

其他的id:

	先哈希求余，

	5个字的:

		根据求余结果 在95-100%当中挑选结果(如75732的测试结果就是100%)

	不是5个字的：

		根据求余结果 在0-99%当中挑选
```

这个是[神人列表](https://gitee.com/ylLty/Five_ID_Num/blob/master/shenRenList.txt)，启动软件时，联网读取，欢迎大家在Gitee上面拉取请求，扩充这个列表
